CSS Cheat Sheet

Basics
Syntax selector{
property: value;
property2: value2;
}
Include external css file <link rel="stylesheet" type="text/css" href="/style.css" />
Internal styles <style type="text/css">
div { color: #444;}
</style>
Inline styles <tag style="property: value"> </tag>
Attribute Selectors
a[target] links with a target attribute
a[target="_blank"] links which open in new tab
[title~="chair"] title element containing a word
Pseudo classes
a:hover link with mouse over it
input:checked checked inputs
p::first-letter first letter in p

Flexbox-Items
order:
Allows you to explictly set the order you want each child to appear in.		 integer
flex-grow:
Is used to specify how much space (and in what proportions) flex items absorb along the main axis.		 1 (applied to all)
(1, 2 and 3)
flex-basis:
Is used to specify the initial size of an element styled with flex-grow and/or flex-shrink.		 50% 25%;
flex-shrink:
Is used to specify how much flex items shrink and in what proportions along the main axis.		 flex-shrink: 1;
align-self:
Sets alignment for individual item. See "align-items" for options.		 flex-start|flex-end|center|baseline|stretch;
